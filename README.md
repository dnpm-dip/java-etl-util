# DNPM:DIP - Java ETL Utilities


**WORK IN PROGRESS**

Java utilities to set up ETL of DIP data exports using the original Scala DTO libraries.
Essentially serves as a wrapper/façade (or "anti-corruption layer"): The various modules provide Java-idiomatic builder methods which encapsulate Scala idioms for DTO instantiation.
